# Personal Portfolio

A personal portfolio website showcasing my projects, technical skills, and experience in software development, AI, and product design.

## Overview

This portfolio was built as a responsive web project to highlight my background, experience, and technical work in a clean and accessible format. The site includes sections for About, Experience, Projects, Contact, and a downloadable resume.

## Features

### Core Features
- Clear section structure with headings and paragraph content across Hero, About, Experience, Projects, and Contact
- External links to LinkedIn, GitHub repositories, and a local resume PDF
- Semantic HTML structure using `main`, `section`, and `footer`
- Accessible **Skip to content** link for keyboard users

### Styling and Layout
- Custom pastel color palette with gradients
- Typography styled using **Google Fonts (Poppins)**
- Responsive design for desktop and mobile screens
- CSS Grid layout for the project gallery
- Styled buttons, tags, and interactive elements with hover states, rounded corners, and soft shadows

### Advanced Features
- Sticky in-page navigation with smooth scrolling
- Active-section highlighting while scrolling
- Interactive project filters using tag “chips”
- Embedded PDF resume with fallback support
- JavaScript enhancements using `IntersectionObserver`

## Accessibility
This project was designed with accessibility in mind:
- Semantic HTML and clear section landmarks
- Keyboard-friendly navigation
- Skip link to main content
- Embedded PDF resume includes fallback options
- Responsive layout and readable spacing
- Color choices were tested to maintain readable contrast with dark text
